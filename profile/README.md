## ✦ 关于我们

第五公理 是一个专注于构建新一代AI应用的团队。

我们的名字，取自欧式几何中的第五公理。历史上，人们并不是通过推翻整个体系来创造新世界，而只是对一条基础公理做了一个简单但关键的调整，最终开启了全新的非欧几何。对我们来说，这不仅是一种灵感，也是一种方法：创造一个新世界，未必需要更复杂的系统，可能只需要一个足够简单、足够本质的改变。

我们相信，AI 时代正在迎来这样的 “第五公理时刻”。过去的世界主要围绕人如何使用工具展开，而未来，一个真正全新的世界，可能建立在 A2A 的基础之上——AI 与 AI 之间能够理解、协作、调用、演化，由此形成过去从未出现过的新秩序、新应用和新生态。

第五公理关注的，正是这种新世界的起点。我们致力于构建面向未来的 AI 应用与基础交互方式，探索 A2A 世界的产品形态、协作机制与价值网络。我们始终认为，人仍然是价值的源头：因为人的需求，技术才有方向；因为服务于人的真实目标，AI 的进化才有意义。但我们同样相信，真正值得期待的下一阶段创新，不只是“人使用 AI”，而是 人与AI 开始形成一个新的世界，并最终为人创造前所未有的价值。
